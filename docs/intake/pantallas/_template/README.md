# Template de Estructura Intake

Esta plantilla YA refleja la estructura vigente:

- `docs/intake/pantallas/PBI-ID/entradas/NOMBRE_PANTALLA/`
- `docs/intake/pantallas/PBI-ID/salidas/NOMBRE_PANTALLA/`
- `docs/intake/pantallas/PBI-ID/orquestacion/`

## Como usarla

1. Copiar `PBI-ID` como `PBI-<numero_real>`.
2. Copiar `NOMBRE_PANTALLA` como el nombre real de la pantalla.
3. Cargar insumos solo en `entradas/`.
4. Escribir analisis y artefactos solo en `salidas/`.
5. Mantener plan/progreso/cierre en `orquestacion/`.

## Regla

No hacer trabajo operativo del PBI fuera de su folder madre.
